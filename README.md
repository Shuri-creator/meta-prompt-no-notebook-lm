# Meta Prompts: Caderno Temático no NotebookLM

> Um projeto prático de aprendizagem ativa explorando técnicas avançadas de 
> engenharia de prompts, curadoria de fontes e organização do conhecimento 
> através do NotebookLM.

---

## 📚 Sobre o Projeto

Este repositório documenta o processo de estudo e desenvolvimento de 
**Meta Prompts** — instruções de nível superior que orquestram sistemas 
multiagentes baseados em IA.

O projeto alinha **pensamento crítico**, **curadoria rigorosa de fontes** 
e **organização sistemática do conhecimento** para criar um caderno temático 
interativo no NotebookLM.

### Objetivo Principal

Entender e documentar técnicas avançadas de engenharia de prompts que 
transformam protótipos artesanais em arquiteturas industriais escaláveis, 
demonstrando não apenas a compreensão técnica, mas também o raciocínio 
por trás dos resultados.

### 2️⃣ **Curadoria de Fontes** (`2-curadoria-fontes.md`)

   - Meta-Prompting Recursivo
   - Squad (Orquestrador, Executor, Auditor)

   - JSON Schema e Pydantic
   - Contratos de dados

4. **Protocolos de Auditoria e Qualidade**

5. **Guia de Implementação e Deploy**
