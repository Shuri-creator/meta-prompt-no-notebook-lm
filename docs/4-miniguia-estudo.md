## 📖 Miniguia de Estudo: Meta Prompts

### Resumo Executivo

Meta Prompts são instruções de nível superior que orquestram sistemas 
multiagentes baseados em IA.

### Os 3 Pilares

#### Pilar 1: Segurança (Hardening)
- Dual LLM Pattern
- Salted Tags
- Zero-Trust Architecture

#### Pilar 2: Orquestração (Meta-Prompting Recursivo)
- Orquestrador (Brain)
- Executor (Micro)
- Auditor (Macro)

#### Pilar 3: Qualidade (Auditoria Automática)
- Fidelidade
- Conformidade Estrutural
- Integridade de Isolamento

### Glossário Técnico

| Termo | Definição |
|---|---|
| Dual LLM Pattern | Usar LLM de baixa permissão para sanitizar inputs |
| Meta-Prompting Recursivo | Orquestrador gera prompts para sub-agentes |
| GSSC Pipeline | Gather → Select → Structure → Compress |
| LLM-as-a-Judge | Usar LLM para auditar outputs de outro LLM |
