## 🔗 Acesso ao Projeto NotebookLM

### Sobre o NotebookLM

NotebookLM é uma ferramenta do Google que permite criar cadernos temáticos 
interativos baseados em fontes de conhecimento. Neste projeto, utilizamos 
o NotebookLM para consolidar aprendizados sobre Meta Prompts e engenharia 
avançada de prompts.

### Acessar o Caderno Público

O caderno temático deste projeto está disponível publicamente no NotebookLM:

**Link:** https://notebooklm.google.com/notebook/95210bf8-3768-4c81-9e5b-04e37f54358b

### Como Usar

1. Clique no link acima
2. Explore as fontes adicionadas (5 documentos técnicos)
3. Use a função "Ask" para fazer perguntas sobre o conteúdo
4. Gere resumos e sínteses automáticas

### Fontes Integradas no NotebookLM

O caderno contém 5 fontes técnicas sobre:

1. **Fundação de Inteligência e Segurança** - Hardening e defesas
2. **Matriz de Orquestração de Fluxos** - Meta-Prompting Recursivo
3. **Registro de Esquemas** - Contratos e validação de dados
4. **Protocolos de Auditoria** - Qualidade e conformidade
5. **Guia de Implementação** - Deploy e otimização

### Funcionalidades Utilizadas

- **Ask:** Fazer perguntas ao caderno
- **Generate:** Criar resumos e sínteses
- **Organize:** Estruturar conhecimento

### Complementaridade

Este repositório GitHub documenta o **processo** de estudo, enquanto o 
NotebookLM contém o **resultado** interativo. Use ambos para uma 
compreensão completa do tema.
